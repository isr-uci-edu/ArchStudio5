xArch/xADL Tools Overview

UCI has developed several tools to the help developers create, manage, and manipulate xArch documents and schemas. These include:

  • Apigen - Java tool to generate the Data Binding Library, below, for current and new schemas.
  • Data Binding Library - Java libraries for creating and manipulating xArch and xADL documents.

xADL Schema

The current list of all ArchStudio schema can be obtained after checking out the ArchStudio source code by going to Navigate | Open Resource ... and typing in "*.xsd". This will identify all xsd schema in the ArchStudio code base. Below are some of the more predominant schema:

  • archlight-3.0.xsd
  • domain-3.0.xsd
  • hints-3.0.xsd
  • implementation-3.0.xsd
  • statechart-1.0.xsd
  • structure-3.0.xsd
  • xadlcore-3.0.xsd
  • xlink.xsd

To create a new xADL schema, take a look at the Apigen tool.
